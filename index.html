Kaptifi Meraki Hackathon 2022

Get customer browsing behavior based on food items purchased. What it does Analyze customer network browsing behavior based on food items purchased. How we built it We started with exploring Foodics API, a famous point of sale used here in Saudi Arabia. Initially, we pulled the raw order data from their API and integrated our splash page using Meraki API for client authentication using the client's mac address submitted. We then compare and pull Meraki Network usage data based on filtered Mac addresses from Meraki API. Accomplishments that we're proud of We had this idea for a long time and this hackathon helped us implement it. We are excited to share new insights with our existing and potential customers who are using Foodics POS and Cisco Meraki. What's next for Kaptifi Meraki Hackathon 2022 We would like to explore Omni channels for POS integrations and provide a deeper level of insights using more filters. Also, we shall be integrating this into kaptifi.com as a new module.
